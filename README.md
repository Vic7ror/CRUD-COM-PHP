# Atividade / Projeto de um CRUD utilizando a linguagem php.

Este é um sistema de CRUD (Create, Read, Update, Delete) para gerenciar usuários.

-------------------------------------

- Aplicativos usados: VsCode, XAMPP, MySQL Workbench. 
- Linguagens: HTML, CSS, PHP, MySQL.
- Também usado: Bootstrap.
